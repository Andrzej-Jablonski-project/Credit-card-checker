-   npm
-   parcel-bundler
-   babel
-   sass
-   prettier
-   autoprefixer
-   tailwindcss

## Automation

Automation is built on the basis of bundler prettier and provides the following functionality.

-   live server
-   minification
-   add source maps
-   add prefixes

## Test

-   jest https://jestjs.io/

```bash
npm run test
```

## Installation

```bash
npm install
```

## How to use

Development project:

```bash
npm start
```

Build a project for publication:

```bash
npm build
```

Clean:

```bash
npm clean
```
